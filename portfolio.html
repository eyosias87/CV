School Administration System

School Administration System

This project is a comprehensive school administration system designed to meet various requirements of a school. The system includes features such as:

  • A menu to display different data requested by the school (Console)
  • An overview of all staff, including names, positions, and years of service, with the ability to add new staff (SQL in SSMS)
  • Storing student information, class assignments, grades for each course, and the teacher who assigned the grade, along with the date (SQL in SSMS)
  • Displaying the number of teachers in different departments (EF in VS)
  • Displaying information about all students (EF in VS)
  • Listing all active courses (EF in VS)
  • Calculating monthly salary expenses for each department (SQL in SSMS)
  • Calculating the average salary for different departments (SQL in SSMS)
  • Creating a stored procedure to return important information about a student based on their ID (SQL in SSMS)
  • Assigning grades to a student using transactions to ensure data integrity (SQL in SSMS)
Technologies used include SQL Server Management Studio, Visual Studio with .NET Framework, C#, and scaffolding methods. The system fulfills all the specified requirements and provides a robust solution for school administration.
